# Closing Tag — Interactive HTML/CSS Project
A playful, interactive webpage that documents a learning journey through web development. Each card represents a key stage of the process, complete with personal reflections, styled images, and interactive “like” buttons with animated heart icons.
## Features
- Responsive Layout – Adapts to different screen sizes using clamp() and media queries.
- Custom Theming – Uses CSS variables for fonts, colors, and background gradients.
- Interactive Likes – Click the heart icon or “Like” button to toggle the liked state with smooth animations.
- Image Filters – Each card features a unique CSS filter (brightness, saturate, sepia, blur, etc.).
- Modal Dialog – A retro-style floppy disk dialog triggered by the “Save to memory” button.
- Custom Fonts – Uses Inter-Variable and PressStart2P for a distinctive visual style.
## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, custom properties, animations, keyframes)
- JavaScript (vanilla)
- SVG icons
